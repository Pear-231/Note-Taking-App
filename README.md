# Note-Taking-App
A WIP note taking app as nothing works like I want.

Backend:
- ASP.NET (C#)

Frontend:
- React + Vite (TypeScript)

## Development Setup and Running
#### Visual Studio (Backend)
- Open the solution file and press F5.

#### Visual Studio Code (Frontend)
- Open the workspace file.
- In the terminal run `cd frontend`.
- If running for the first time, run `npm install`.
- Press F5 or run `npm run dev`.
